# emsdoc

## 설치

- 설치 : `npm i docsify-cli -g`
- 초기화 : `docsify init` (폴더 생성 후 해당 폴더에서 위 커맨드를 실행)
- 서버운영 : `docsify serve ./docs`
- 확인 : [http://localhost:3000](http://localhost:3000)

## 실행

일반

- `npm i docsify-cli -g` - 설치가 되지 않은 경우 설치
- `docsify serve ./docs`
